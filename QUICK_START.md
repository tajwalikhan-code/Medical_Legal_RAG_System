# 🚀 QUICK START GUIDE - RAG System Project

## ⚡ 5-Minute Setup

### 1️⃣ Upload to Colab (30 seconds)
```
1. Go to: https://colab.research.google.com/
2. File → Upload notebook
3. Select: Medical_Legal_RAG_System.ipynb
```

### 2️⃣ Run Everything (3 minutes)
```
Press: Runtime → Run all
OR: Ctrl+F9 (Windows) / Cmd+F9 (Mac)

Wait for all cells to complete (look for ✓ symbols)
```

### 3️⃣ Download Files (1 minute)
```
Click folder icon 📁 in sidebar
Right-click each file → Download:
  ✓ Medical_Legal_RAG_System.ipynb
  ✓ RAG_System_Report.txt
  ✓ results_summary.json
  ✓ rag_performance_metrics.png
```

### 4️⃣ Submit to Classroom (1 minute)
```
1. Create ZIP of all files
2. Go to Google Classroom assignment
3. Click "Add or create" → Upload
4. Select ZIP file
5. Click "Turn in"
```
